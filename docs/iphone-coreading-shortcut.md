# iPhone 快捷指令：发送到 Nikto 共读

不在仓库写入真实 Home 地址。部署后打开共读页，点“设置 iPhone 分享到共读”里的“复制这个共读入口”，得到当前受保护入口。

1. 打开 iPhone“快捷指令”，新建快捷指令，名称设为“发送到 Nikto 共读”。
2. 点快捷指令详情，开启“在共享表单中显示”，接收类型选择“URL”和“网页”。
3. 添加“URL”动作：先粘贴共读入口，再把“快捷指令输入”变量接在末尾。
4. 添加“打开 URL”动作，输入使用上一步的 URL。
5. 保存。之后在 Chrome 打开公开章节，点分享，选择“发送到 Nikto 共读”。

快捷指令只传当前公开网页 URL，不传 Home 密码、token、正文、批注或记忆。页面仍须通过现有 Home 身份认证。
